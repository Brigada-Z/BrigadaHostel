# Carpeta de Paginas del Panel

Aca estan metidos todos los archivos que arman el panel de control del hostal. Cada uno se encarga de una parte distinta para que no sea un lio de codigo.

### Que hay en cada archivo:

* **resumen.html**: Es la pantalla principal. Aca se ve de un vistazo como viene el negocio, la ocupacion de la semana y cuanta plata esta entrando.
* **reservas.html**: Aca se maneja todo el movimiento de las señas y los ingresos. Podes ver quien entra hoy, quien sale y de donde vienen las reservas (si por Booking, Airbnb o directo).
* **habitaciones.html**: Este es el mapa del hostal. Te muestra que piezas estan ocupadas, cuales hay que limpiar y si alguna esta rota o en mantenimiento.
* **huespedes.html**: El listado de la gente. Sirve para saber quien nos visita mas seguido, de que pais vienen y si estan contentos con el servicio.
* **configuracion.html**: Aca se toca la parte interna. Datos del hostal, el IVA, la moneda que usamos y las claves del sistema.

Todo esto usa el mismo diseño que esta en la carpeta de estilos para que no se vea distinto cuando pasas de una pestaña a otra.
