# Dashboard Hotelero - Gestión de Hotel

## Introducción

Este proyecto académico corresponde al módulo Full Stack I. La maqueta propone un sistema de gestión hotelera para hoteles pequeños y medianos de la Ciudad de Córdoba.

El problema principal que busca resolver es la dificultad que genera la gestión manual de reservas, habitaciones y servicios adicionales. La propuesta plantea una interfaz clara, centralizada y responsive.

## Funcionalidades representadas en la maqueta

- Home / Landing Page con presentación principal del sistema.
- Barra de navegación con enlaces a Inicio, Quiénes Somos.
- Sección visual con propuesta de valor del sistema.
- Página institucional "Quiénes Somos".

## Tecnologías utilizadas

- HTML5
- CSS3

No se utiliza JavaScript, librerías ni frameworks.

## Cómo ejecutar la maqueta

1. Descargar o clonar el repositorio.
2. Ingresar a la carpeta `maqueta`.
3. Abrir el archivo `index.html` en el navegador.
4. Para ver la página institucional, abrir `quienes-somos.html` o navegar desde el menú.
